# Cpp-Simple-Games

Simple project to create a game various games in C++

## Setup

Clone project
cd into project folder

### Linux
```
sudo apt-get install libsdl2-dev
mkdir build
cd build
cmake ..
make
./Games
```

## Games
### Tic-Tac-Toe
Simple Game of Tic-Tac-Toe with an AI that will play optimally
